# Vogue Point Telegram Bot

Telegram-бот для управления и отображения ассортимента одежды.  

## Описание

Бот позволяет:
- Просматривать каталог одежды (по категориям, размерам, цветам).
- Получать информацию о товаре (цена, наличие, описание).

## Технологии

- **Python** 3.10+
- **python-telegram-bot** (v20.x) — библиотека для работы с Telegram Bot API
- **SQLite** / PostgreSQL — хранение данных о товарах, пользователях, заказах
- **python-dotenv** — управление переменными окружения
- **Git** — контроль версий

## Команды бота

Команда	Описание
/start	Приветствие и главное меню
/catalog	Показать каталог одежды
/filters	Открыть фильтры (категория, размер, цвет)
/item [ID]	Показать карточку товара

## Примеры взаимодействия

Пользователь отправляет /catalog → выбирает категорию (например, "Футболки") → получает список товаров → нажимает на товар → видит цену, наличие, описание и фото.
